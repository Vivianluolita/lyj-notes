### 页面布局没，三栏布局，左右宽度300px，中间自适应

  1. 快捷键 section.layout.float
    ```
      <section class="layout float"></section>
    ```
  2. 快捷键 div.left+div.center+div.left
    ```
      <div class="left"></div>
      <div class="right"></div>
      <div class="center"></div>
    ```
  3. 五种方法 （3-3）
    ```
      浮动解决方法
      绝对定位解决方法
      flexbox解决方法
      表格解决方法
      网格解决方法
    ```

  4. 优缺点
    ```
      浮动兼容性比较好
      绝对方便，但是都是脱离文档流
      flex手机
      表格布局兼容性很好，当其中一个单元格高度超出时候，其他也需要超出
    ```
