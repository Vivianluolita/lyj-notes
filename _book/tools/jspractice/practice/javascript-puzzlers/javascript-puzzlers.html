reduce()

array.reduce(function(total, currentValue, currentIndex, arr), initialValue) 计算数组元素相加后的总和：125

var numbers = [65, 44, 12, 4];

function getSum(total, num) {
    return total + num;
}
function myFunction(item) {
    document.getElementById("demo").innerHTML = numbers.reduce(getSum);
}

JavaScript Infinity 属性

Infinity 属性用于存放表示正无穷大的数值。 无法使用 for/in 循环来枚举 Infinity 属性，也不能用 delete 运算符来删除它。Infinity 不是常量，可以把它设置为其他值。 Infinity % 2 gives NaN

该值介于 2 ~ 36 之间。
如果省略该参数或其值为 0，则数字将以 10 为基础来解析。如果它以 “0x” 或 “0X” 开头，将以 16 为基数。
如果该参数小于 2 或者大于 36，则 parseInt() 将返回 NaN。

数组比较

    var a = [1, 2, 3],
        b = [1, 2, 3],
        c = [1, 2, 4]
    a ==  b false
    a === b false
    a >   c false
    a <   c true

数组按字典顺序与>和<进行比较，但不与==和===进行比较
